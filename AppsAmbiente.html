Aplicación Ambiente

Esta aplicación tiene como objetivo facilitar al usuario la carga de datos de ambiente, como lo pueden ser los KPI, residuos, agua, energía y la posibilidad de modificar datos anuales como el SSU. Como extra tiene la posibilidad de extraer datos a partir de una imagen para copiarla y pegarla, esto con el objetivo de reducir el error humano. A partir de estos datos salen diferentes gráficos los cuales permiten la visualización de datos para la fácil toma de decisiones y estar al tanto actualizado día a día sobre las cargas realizadas.
